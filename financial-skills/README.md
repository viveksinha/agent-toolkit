# financial-skills

**Portfolio analysis and refresh workflows** — brokerage statement parsing and equity canvas updates.

Skills in this plugin use **generic placeholders** only. Configure paths, tickers, and cluster definitions locally. Never commit real holdings, account numbers, or export files (see [no-pii-push](../engineering-skills/no-pii-push/SKILL.md)).

## Skills

| Skill | What it does |
|-------|-------------|
| [ibkr-portfolio-analysis](skills/ibkr-portfolio-analysis/SKILL.md) | Parse an IBKR Activity Statement CSV and render a portfolio analysis canvas |
| [indian-stock-portfolio-refresh](skills/indian-stock-portfolio-refresh/SKILL.md) | Refresh valuation, risk, macro, and daily-actions canvases from a Trendlyne Excel export |
