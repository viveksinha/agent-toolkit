# Kinder messages

## When to apply

Whenever the user asks for message text, a rephrase, or tone help **toward another person** (coworker, customer, partner), default to this skill—even if they only say "draft this" or "fix this email."

## Principles

1. **Assume good intent** — Frame asks as invitations, not audits ("when you have a moment" beats "you need to").
2. **Reduce pressure** — Offer flexibility (alternate time, optional pre-read, "no rush" where honest).
3. **Collaborative voice** — Use "we" where accurate; acknowledge their time and constraints.
4. **Clear but soft** — Keep deadlines and facts; soften *how* they are delivered (appreciation, one question instead of a list of demands).
5. **Avoid** — Passive aggression, guilt ("I already asked"), all-caps emphasis, "just," excessive exclamation points, or empty filler that reads as insincere.

## Workflow

1. If the user pasted a draft, **rephrase it**; do not only critique unless they asked for feedback only.
2. Offer **1–3 variants** when useful: brief ping, standard, extra-accommodating.
3. Preserve **non-negotiables**: real deadlines, legal/compliance wording, or technical precision they specified—only adjust tone around them.
4. Match **register** to channel: Slack/DM slightly warmer and shorter; email slightly more complete; executive updates stay crisp but respectful.

## Quick patterns

| Harsher | Kinder |
|--------|--------|
| You need to X before Y | If you can X before Y, that would help; if not, we can adjust Y |
| Why wasn't this done? | I'm trying to understand what blocked this—happy to pair if useful |
| Reminder: deadline is Friday | Friendly heads-up that we're aiming for Friday—shout if that's tight |

## Examples

**User:** Rephrase: "Soumya, go through these before the call tonight or move the call."

**Response:** Give a short option such as: *Hi Soumya — if you get a chance to skim these before our call tonight, great; if you'd rather have more time, happy to reschedule—whatever works better for you.*

---

Keep the skill **silent in the reply**—apply the behavior; do not announce "following the kinder-messages skill."
