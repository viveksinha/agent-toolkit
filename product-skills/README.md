# product-skills

**Product management skills — discover problems, write specs, design experiments, and ship with confidence.**

Each skill is a standalone prompt template. Commands chain skills into guided workflows.

## Skills

| Skill | What it does |
|-------|-------------|
| [7-powers](skills/7-powers/SKILL.md) | Analyze competitive advantage using Hamilton Helmer's 7 Powers framework |
| [craft-spec](skills/craft-spec/SKILL.md) | Turn messy ideas into a clear, structured PRD |
| [craft-experiment-design](skills/craft-experiment-design/SKILL.md) | Write a hypothesis, define success metrics, and plan a holdout strategy |
| [craft-experiment-readout](skills/craft-experiment-readout/SKILL.md) | Summarize results, call a winner, and draft a stakeholder-ready recommendation |
| [craft-discovery-synthesis](skills/craft-discovery-synthesis/SKILL.md) | Extract themes and insights from raw user interviews and feedback |
| [create-user-stories](skills/create-user-stories/SKILL.md) | Break a PRD into implementable user stories with acceptance criteria |
| [draft-spec](skills/draft-spec/SKILL.md) | Draft a detailed spec with problem context, solution design, and strategic framing |
| [jobs-to-be-done](skills/jobs-to-be-done/SKILL.md) | Uncover functional, social, and emotional jobs driving customer behavior |
| [pre-mortem](skills/pre-mortem/SKILL.md) | Imagine launch failure and work backward to identify risks before you ship |
| [prioritize](skills/prioritize/SKILL.md) | Take a list of ideas or initiatives and quickly rank them using RICE or another framework |
| [competitor-monitoring](skills/competitor-monitoring/SKILL.md) | Visit competitor sites and write a structured competitive intelligence report |
| [market-research-digest](skills/market-research-digest/SKILL.md) | Synthesize Drive files, meeting notes, and web signals into a weekly research summary |
| [what-would-lenny-do](skills/what-would-lenny-do/SKILL.md) | Answer product strategy and growth questions using Lenny Rachitsky's archive (requires lennysdata MCP) |

## Commands

| Command | What it does |
|---------|-------------|
| [write-prd](commands/write-prd.md) | Guided PRD writing — gathers context conversationally, then generates a structured spec |
| [run-discovery](commands/run-discovery.md) | Full discovery workflow — synthesize research, identify assumptions, design experiments |
