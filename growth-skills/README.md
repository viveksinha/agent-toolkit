# growth-skills

**Growth skills — activation analysis, retention strategies, funnel optimization, and go-to-market planning.**

| Type | Name | What it does |
|------|------|-------------|
| Skill | [build-metric-tree](skills/build-metric-tree/SKILL.md) | Decompose a top-line metric into a sized tree, identify where the real leverage is and where it isn't |
| Skill | [diagnose-activation](skills/diagnose-activation/SKILL.md) | Diagnose where new users fail to activate, identify the aha moment, and build a sized plan to move activation rate |
| Skill | [diagnose-retention](skills/diagnose-retention/SKILL.md) | Decompose your retention curve into cohorts, identify retention-predictive behaviors, and build a plan to bend the curve |
| Skill | [diagnose-monetization](skills/diagnose-monetization/SKILL.md) | Find revenue leaks, evaluate packaging and pricing, and size expansion revenue opportunities |
| Skill | [diagnose-acquisition](skills/diagnose-acquisition/SKILL.md) | Map where users come from, measure channel quality and CAC, and find the highest-leverage acquisition channels |
| Skill | [map-growth-loops](skills/map-growth-loops/SKILL.md) | Identify and measure self-reinforcing growth loops (viral, content, paid, usage), find where loops break, and invest in compounding growth |
